적용하기 이전에 @awk계정에 해당 채널의 관리 권한을 부여하세요.
"감사 / 삭제 / 수정 / 차단" 권한이 필요합니다.
각각 [오래된 글 수정시 자동삭제], [삭제] , [격리], [차단] 기능을 위해 필요합니다.
만일 필요가 없다면 해당 권한을 제외하고 부여하면 됩니다.
단, 예외적으로 삭제 권한은 권한 여부를 확인할 때 필요하므로 반드시 설정해 주셔야 합니다.

한 번 설정한 값은 해제하기 전까지 수정할 수 없습니다.
만일 해제를 원한다면 권한을 전부 해제한 다음 게시판 ID를 입력하고 삭제 버튼을 눌러 삭제할 수 있습니다.

게시글 격리 기능을 적용했다면, "운영"이라는 이름의 탭이 있어야 합니다.
설정된 숫자보다 추천 - 비추천 수가 적은 게시글들은 자동으로 운영 탭으로 이동하게 됩니다.
